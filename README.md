# 乃敢与君绝

> 纵马踏花去，仗剑走天涯

基于 [Hexo 7.3](https://hexo.io/) 搭建的个人技术博客，使用 [Butterfly](https://butterfly.js.org/) 主题。

## 关于本站

博客主要记录安全研究、CTF 题解、工具使用教程等内容，托管于 GitHub Pages。

## 技术栈

- **框架**: Hexo 7.3
- **主题**: Butterfly 5.5
- **代码高亮**: highlight.js (darker 主题)
- **评论**: 待集成
- **部署**: GitHub Pages

## 本地开发

```bash
git clone git@github.com:shen060606/hexo-blog.git
cd hexo-blog
npm install
npx hexo server
```

浏览器访问 `www.abyssa1.com` 即可预览。
